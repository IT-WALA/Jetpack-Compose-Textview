# Jetpack-Compose-Textview

*Various textview examples including
1) Text color
2) Font size
3) Font family
4) Bold, underline, italic
5) Text alignment
6) Letter spacing
7) Max lines
8) Text overflow & wrap
9) Style property
10) Padding

*Since so many textviews are used, the column is made vertically scrollable. This is equivalent to the scrollview.
